# 基本命令
  1. 项目构建命令
    ```
    yarn create vite user.ui --template vue-ts
    ```
  2. 运行项目
    ```
    yarn run dev | yarn vite
    ```
  3. 安装依赖
    1. npm install || yarn
      根据package.json安装依赖
    2. npm install <module_name> <--save || ''>
      将指定依赖保存到dependencies，以便生产环境部署
    3. 3.2 + <--save-dev || -D>
      将指定依赖保存到devDependencies，代表仅部署开发环境

  3. 问题及知识点记录放在根md下
  4. 为了方便GitHub上的预览，所有md均命名为README.md
